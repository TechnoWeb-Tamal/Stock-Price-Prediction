## 📌 Overview
The Real-Time Stock Price Predictor is a dynamic web application that forecasts stock market trends. Unlike static models, this application integrates the `yfinance` library to fetch and process live market data.

This project demonstrates a full-stack approach to data science by combining a robust Flask backend with a custom-styled frontend.

## ✨ Features
* **Live Data Integration:** Fetches real-time stock data via the Yahoo Finance API.
* **Data Visualization:** Generates trend graphs using Matplotlib for better market analysis.
* **Dynamic Forecasting:** Processes current market variables for trend predictions.
* **Custom UI:** Features a dedicated HTML/CSS frontend managed by Flask.

## 🛠️ Tech Stack
* **Backend:** Flask (Python)
* **Data Analysis:** Pandas, NumPy
* **Visualization:** Matplotlib
* **API:** `yfinance` (Yahoo Finance)
* **Frontend:** HTML5, CSS3
